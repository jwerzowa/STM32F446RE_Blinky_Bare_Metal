# STM32F446RE_Custom_Drivers

Playing around with embedded at register level, building personal drivers for the STM32F446RE Nucleo board.

Will function as a template for easy prototyping with the board.

Currently working on RCC and GPIO driver.

Next Drivers to be worked on:
- SysTick
- UART
- I2C
- SPI

 Will be eventually used to explore CAN protocol at a later date.
